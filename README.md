# 📂 File Organizer

Um organizador inteligente de arquivos desenvolvido em Python.

Organize automaticamente seus arquivos por tipo, detecte duplicados, acompanhe o histórico das movimentações e monitore pastas em tempo real.

![Python](https://img.shields.io/badge/Python-3.12-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-orange)

---

## ✨ Funcionalidades

- 📁 Organização automática por categoria
- 📄 Suporte a mais de 100 extensões
- 🔍 Detecção de arquivos duplicados
- 📊 Relatório das movimentações
- ↩️ Desfazer última organização
- 👀 Modo Simulação (Preview)
- ⚡ Monitoramento automático da pasta Downloads
- 📝 Histórico de execuções

## 📦 Instalação

Clone o projeto:

```bash
git clone https://github.com/BrunnoAlves0011/file-organizer.git
```

Entre na pasta:

```bash
cd file-organizer
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute:

```bash
python app.py
```
